# Μοντέλα Ανοιχτού Βάρους (Open-Weight Models)

## Ορισμός

Τα **μοντέλα ανοιχτού βάρους** είναι μοντέλα που μπορούν να **εκτελεστούν τοπικά χωρίς περιορισμούς ασφαλείας**. Αυτά τα μοντέλα είναι προσβάσιμα για λήψη και εκτέλεση σε τοπικές υποδομές.

## Χαρακτηριστικά

### Βασικές Ιδιότητες

- **Τοπική Εκτέλεση**: Μπορούν να εκτελεστούν χωρίς σύνδεση σε εξωτερικό API
- **Εύκολη Τροποποίηση**: Επιτρέπουν fine-tuning χωρίς περιορισμούς
- **Αδύναμοι ή Χωρίς Μηχανισμούς Ασφαλείας**: Συχνά δεν διαθέτουν άρνηση σε επιβλαβείς αιτήσεις
- **Χαμηλότερες Ικανότητες**: Γενικά ασθενέστερα από τα frontier μοντέλα

### Γνωστά Παραδείγματα

- Qwen2.5-7B
- Mixtral-8x22B
- Qwen2.5-72B
- Llama-3.1-405B

## Ρόλος στις Επιθέσεις

Τα μοντέλα ανοιχτού βάρους χρησιμοποιούνται σε **συνδυασμένες επιθέσεις**:

1. **Αφαίρεση Περιορισμών**: Μπορούν να fine-tune για αφαίρεση μηχανισμών άρνησης
2. **Σύνθεση Απαντήσεων**: Χρησιμοποιούνται για σύνθεση απαντήσεων από frontier μοντέλα
3. **Κατανομή Επιθέσεων**: Επιτρέπουν διάσπαση επιβλαβών εργασιών
