# Jailbreaks (Αναδυόμενες Φρουρές)

## Επισκόπηση

Οι **αναδυόμενες φρουρές** (jailbreaks) είναι τεχνικές που παρακάμπτουν τους μηχανισμούς άρνησης των μοντέλων για να αποσπάσουν επιβλαβές περιεχόμενο. Αποτελούν κλασικές επιθέσεις που στοχεύουν στην άμεση απόκτηση απαγορευμένων πληροφοριών.

## Τύποι Jailbreaks

### Μονής Στροφής (Single-Turn)
- **Adaptive Attacks**: Απλές προσαρμοστικές επιθέσεις
- **PAIR**: Επιθέσεις με αμφίδρομη αλληλεπίδραση
- **Adversarial Reasoning**: Επιθέσεις με αντίθετη συλλογιστική

### Πολλαπλών Στροφών (Multi-Turn)
- **Crescendo**: Σταδιακή κλιμάκωση πολύπλοκων αιτημάτων

## Περιορισμοί

Τα jailbreaks συχνά:
- Παράγουν μη ενημερωτικές απαντήσεις
- Ανιχνεύονται εύκολα από φίλτρα ασφαλείας
- Είναι εύκολα ανιχνεύσιμα από API εποπτείας
- Πέφτουν κοντά σε απορριπτόμενες ερωτήσεις BSD στην UMAP προβολή
