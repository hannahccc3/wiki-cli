# R2D2-7B

## Επισκόπηση

Το R2D2-7B είναι ένα μοντέλο που εκπαιδεύτηκε με adversarial training για να είναι ανθεκτικό σε επιθέσεις jailbreaking. Αναπτύχθηκε από το CAIS (Center for AI Safety).

## Αποτελέσματα Επιθέσεων

| Μέθοδος | Ποσοστό Επιτυχίας |
|---------|-------------------|
| GCG | 6% |
| PAIR | 48% |
| TAP | 61% |
| Prompt | 8% |
| Prompt + RS + Self-Transfer | 12% |
| **In-context Prompt + Random Search** | **100%** |

## Βασικά Ευρήματα

- Το adversarial training (αντί για απλή ευθυγράμμιση) παρέχει κάποια προστασία
- Η τεχνική in-context learning είναι ιδιαίτερα αποτελεσματική
- Το self-transfer δεν λειτουργεί για αυτό το μοντέλο
