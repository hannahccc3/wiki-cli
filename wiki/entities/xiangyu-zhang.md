# Xiangyu Zhang

## Overview

Xiangyu Zhang is a researcher and faculty member at Purdue University's Computer Science department. He is a co-author of the RL-JACK paper and a leading figure in software security and machine learning research.

## Research Contributions

Zhang contributed to the RL-JACK project with expertise in software security and adversarial machine learning. His research focuses on understanding and improving the security properties of machine learning systems.
